Malware Encyclopedia

Learn about different types of malware and how to protect yourself

Virus

Self-replicating code that attaches to programs

Trojan

Disguises itself as legitimate software

Ransomware

Encrypts files and demands payment

Spyware

Secretly collects user data and activity

Adware

Forces unwanted advertisements on the user

Worm

Spreads across networks without user action

Rootkit

Hides deep in OS, grants persistent access

Keylogger

Records every keystroke, stealing passwords

Botnet

Network of infected machines controlled remotely

Phishing

Tricks users into revealing sensitive information
