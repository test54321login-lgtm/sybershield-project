Adware

Low Severity

What is Adware?

Adware is software that automatically displays or downloads advertising material when a user is online. While not always malicious, adware can become problematic when it is installed without the user's consent or when it displays unwanted, inappropriate, or excessive advertisements. Adware often comes bundled with free software or is installed through malicious websites.

How Does it Work?

  1. Adware is installed on a device, often bundled with other software
  2. It monitors user's browsing habits and preferences
  3. Displays targeted advertisements based on collected data
  4. May redirect web searches to advertising websites
  5. Can slow down the system and consume bandwidth

Real-World Example

Fireball was a widespread adware program discovered in 2017 that hijacked web browsers and turned them into puppets for ad fraud. It affected over 250 million computers worldwide, manipulating browser settings to control user web traffic and install additional software. Fireball had the capability to act as a full-featured browser hijacker, redirecting traffic to malicious sites.

How to Protect Yourself

  • Be cautious when installing free software and opt-out of additional offers
  • Use ad blockers to prevent unwanted advertisements
  • Keep your browser and security software updated
  • Regularly review and remove unfamiliar browser extensions
  • Download software only from official or trusted sources
  • Use anti-malware software that detects adware
